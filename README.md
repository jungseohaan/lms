# LMS

***

## 프로젝트명
LMS

## 프로젝트 설명
이 프로젝트는  Spring Data JPA와 Spring Data REST를 활용하여 간단한 형태의 API를 만들고, 복잡한 API는 QueryDSL을 활용하여 개발
